Qualification et suivi de vos leads

Moyen 25 min CRM

Introduction au module Pré-Opportunité

Le module Pré-Opportunité permet de gérer efficacement la qualification et le suivi de vos leads dans Dolibarr. Il constitue la première étape du processus commercial, permettant de filtrer et qualifier les contacts avant leur transformation en opportunités commerciales.

À savoir

Ce module est essentiel pour la gestion quotidienne de vos activités commerciales. Il permet d'organiser vos contacts entrants et de prioriser vos actions commerciales en fonction de critères de qualification précis.

Objectifs du module

  • Centraliser tous les leads entrants dans un espace dédié
  • Qualifier rapidement les prospects selon des critères définis
  • Prioriser les actions commerciales
  • Faciliter la conversion vers des opportunités commerciales
  • Améliorer le taux de transformation commercial

Configuration du module

Avant d'utiliser le module Pré-Opportunité, il est nécessaire de le configurer correctement. Voici les étapes à suivre :

1

Activer le module

Rendez-vous dans Configuration → Modules. Cherchez "Pré-Opportunité" dans la liste des modules disponibles et activez-le en cliquant sur le bouton d'activation.

2

Paramétrer les options

Configurez les options selon vos besoins métier : critères de qualification, statuts personnalisés, champs obligatoires, etc. Cette personnalisation permet d'adapter le module à vos processus commerciaux spécifiques.

3

Définir les permissions

Attribuez les droits d'accès appropriés à vos utilisateurs dans Utilisateurs → Groupes. Définissez qui peut créer, modifier ou supprimer des pré-opportunités.

4

Personnaliser les workflows

Configurez les étapes de qualification et les actions automatiques associées à chaque changement de statut pour automatiser votre processus commercial.

Attention

La configuration initiale doit être validée par un administrateur. Une fois le module en production, les modifications de configuration peuvent impacter les données existantes.

Utilisation pratique

Créer une pré-opportunité

Pour créer une nouvelle pré-opportunité :

  1. Accédez au menu CRM → Pré-Opportunités
  2. Cliquez sur le bouton "Nouvelle pré-opportunité"
  3. Renseignez les informations obligatoires : nom du contact, entreprise, source du lead
  4. Ajoutez les informations complémentaires selon vos critères de qualification
  5. Validez la création

Qualifier un lead

La qualification permet d'évaluer le potentiel commercial d'un lead :

  • Budget : Le prospect dispose-t-il d'un budget identifié ?
  • Autorité : Êtes-vous en contact avec un décideur ?
  • Besoin : Le besoin est-il clairement exprimé ?
  • Timing : Quel est le calendrier de décision ?

Méthode BANT

La méthode BANT (Budget, Authority, Need, Timing) est recommandée pour qualifier efficacement vos leads et prioriser vos actions commerciales.

Convertir en opportunité

Une fois le lead qualifié, vous pouvez le convertir en opportunité commerciale :

  1. Ouvrez la fiche de la pré-opportunité
  2. Vérifiez que tous les critères de qualification sont remplis
  3. Cliquez sur "Convertir en opportunité"
  4. Le système crée automatiquement une opportunité liée et un projet associé

Important

Vérifiez toujours vos données avant validation. Une fois convertie, la pré-opportunité ne peut plus être modifiée directement.

Bonnes pratiques

Pour optimiser votre utilisation du module Pré-Opportunité, suivez ces recommandations :

  • Maintenez vos données à jour régulièrement : Actualisez les statuts et les informations dès que vous obtenez de nouvelles informations sur un lead
  • Utilisez les filtres pour gagner du temps : Créez des vues personnalisées pour accéder rapidement aux leads prioritaires
  • Formez votre équipe aux fonctionnalités avancées : Assurez-vous que tous les commerciaux maîtrisent les critères de qualification
  • Définissez un délai de traitement : Fixez un SLA pour le traitement des pré-opportunités (ex : qualification sous 48h)
  • Suivez vos KPIs : Taux de conversion, délai moyen de qualification, nombre de leads par source
  • Nettoyez régulièrement : Archivez ou supprimez les leads obsolètes pour maintenir une base propre
  • Consultez le support en cas de doute : L'équipe Cyber-Technique est là pour vous aider

Astuce Pro

Créez des templates de qualification pour standardiser le processus et améliorer la cohérence des évaluations entre les différents commerciaux de votre équipe.
