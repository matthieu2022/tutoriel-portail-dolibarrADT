Gestion des plateformes LMS

Guide complet pour l'administration et l'utilisation de nos Learning Management Systems

Introduction aux LMS ADT

L'Académie du Tourisme utilise plusieurs plateformes LMS (Learning Management System) basées sur Moodle pour gérer et diffuser ses formations en ligne. Ces plateformes sont essentielles pour :

  • Héberger et organiser les contenus pédagogiques
  • Suivre la progression des apprenants
  • Gérer les inscriptions et les sessions de formation
  • Faciliter les interactions entre formateurs et apprenants
  • Délivrer les attestations et certificats de formation

Important pour les commerciaux

Les plateformes LMS sont au cœur de notre offre de formation. Il est essentiel de comprendre leur fonctionnement pour présenter efficacement nos formations aux prospects et accompagner les clients dans leur parcours.

Nos différentes plateformes LMS

L'ADT gère plusieurs instances Moodle, chacune dédiée à des formations ou publics spécifiques :

LMS Principal ADT

URL : learning.academie-tourisme.fr

Usage : Formations principales BtoB et BtoC

Nombre d'apprenants : ~5000 utilisateurs actifs

LMS Hospitality School

URL : hospitality-school.academy

Usage : Formations secteur hôtellerie-restauration

Nombre d'apprenants : ~2000 utilisateurs actifs

LMS Tourism School

URL : tourism-school.academy

Usage : Formations secteur tourisme et voyages

Nombre d'apprenants : ~3000 utilisateurs actifs

LMS Entreprises

URL : corporate.academie-tourisme.fr

Usage : Formations sur-mesure pour entreprises

Nombre d'apprenants : Variable selon contrats

Gestion quotidienne des plateformes

Accès administrateur

Pour accéder aux fonctions d'administration d'une plateforme LMS :

  • Se connecter avec un compte administrateur
  • Cliquer sur "Administration du site" dans le menu
  • Accéder aux différentes sections de gestion

Tâches courantes

Création de cours

  • Définir la structure du cours
  • Importer les contenus pédagogiques
  • Configurer les paramètres
  • Tester le parcours apprenant

Gestion des utilisateurs

  • Créer des comptes apprenants
  • Gérer les inscriptions aux cours
  • Attribuer les rôles (formateur, tuteur...)
  • Suivre l'activité des utilisateurs

Suivi pédagogique

  • Consulter les rapports de progression
  • Analyser les résultats aux évaluations
  • Identifier les apprenants en difficulté
  • Générer les attestations de formation

Maintenance technique

  • Effectuer les mises à jour Moodle
  • Gérer les plugins et extensions
  • Optimiser les performances
  • Sauvegarder régulièrement les données

Bonnes pratiques

Toujours tester les modifications sur un cours test avant de les appliquer aux cours en production. Effectuer des sauvegardes avant toute mise à jour importante.

Guide pour les commerciaux

Présenter nos plateformes LMS aux prospects

Voici les points clés à mettre en avant lors de la présentation de nos LMS :

  • Accessibilité 24/7 : Les apprenants peuvent se connecter à tout moment, de n'importe où
  • Interface intuitive : Navigation simple et ergonomique, adaptée à tous les profils
  • Suivi personnalisé : Tableaux de bord complets pour suivre la progression
  • Contenus riches : Vidéos, documents, quiz, exercices interactifs
  • Support technique : Équipe cyber-technique disponible pour assistance

Démonstration type

Pour une démonstration efficace :

  1. Montrer la page d'accueil et la navigation
  2. Parcourir un exemple de cours complet
  3. Démontrer les outils collaboratifs (forums, messagerie)
  4. Présenter le tableau de bord apprenant
  5. Expliquer le processus de certification

Astuce commerciale

Proposez toujours un accès démo temporaire au prospect. Rien ne vaut l'expérience directe pour convaincre !
