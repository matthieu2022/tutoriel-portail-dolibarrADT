Introduction au déploiement Dolibarr V1

Ce document détaille le fonctionnement du logiciel Dolibarr pour la gestion de la relation client au sein de l'Académie du Tourisme. Il définit précisément le cycle de vie des contacts, distinguant les suspects, les prospects de différents degrés et les clients ayant déjà finalisé un contrat. Le guide explique également comment identifier les doublons par un code couleur spécifique afin de maintenir une base de données propre. Les utilisateurs y trouvent des instructions pour suivre l'avancement des projets BtoB, depuis l'expression du besoin jusqu'à la signature finale. Enfin, l'outil propose des conseils pratiques sur l'utilisation des filtres de recherche, de l'historique des échanges et de la barre d'outils.

Pour rappel : comment se connecter ?

Se connecter à Dolibarr

Pour accéder à votre espace Dolibarr, rendez-vous sur l'URL suivante :

Lien de connexion

URL : https://gestion.academie-tourisme.fr/

  1. Identifiant : Votre adresse email professionnelle ADT
  2. Mot de passe : Le mot de passe qui vous a été communiqué par la Cyber-Technique
  3. Cochez la case "Se souvenir de moi" si vous utilisez un ordinateur personnel
  4. Cliquez sur "Connexion"

Mot de passe oublié ?

Si vous avez oublié votre mot de passe, utilisez le lien "Mot de passe oublié" sur la page de connexion, ou contactez l'équipe Cyber-Technique via le système de ticketing.

Découvrir l'interface

Une fois connecté, vous arrivez sur le tableau de bord. Prenez le temps de vous familiariser avec les différentes zones :

  • Barre de navigation supérieure : Accès rapide aux modules principaux (Tiers, Projets, Commerce, etc.)
  • Menu latéral gauche : Navigation détaillée avec tous les sous-menus
  • Zone centrale : Contenu principal et widgets de votre tableau de bord
  • Barre de recherche : En haut à droite, pour rechercher rapidement clients, projets, factures...
  • Menu utilisateur : En haut à droite, pour accéder à votre profil et aux paramètres

Astuce de navigation

Vous pouvez personnaliser votre tableau de bord en ajoutant ou supprimant des widgets. Cliquez sur l'icône de configuration (roue dentée) pour accéder aux options.
