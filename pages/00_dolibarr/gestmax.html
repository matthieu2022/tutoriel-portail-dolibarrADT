Qu'est-ce que GestMax ?

GestMax est un logiciel de gestion administrative spécialisé dans le suivi des dossiers de formation, le traitement des financements (OPCO, CPF, Pôle Emploi) et la gestion réglementaire. L'intégration avec Dolibarr permet de centraliser les informations commerciales et administratives pour un suivi complet du parcours client.

Pourquoi intégrer GestMax ?

GestMax et Dolibarr répondent à des besoins complémentaires : Dolibarr gère la relation client et le commercial, GestMax gère l'administratif et les financements. L'intégration évite la double saisie et garantit la cohérence des données.

Avantages de l'intégration

Synchronisation

Échange bidirectionnel des données entre les deux systèmes.

Zéro double saisie

Les informations saisies une fois sont disponibles partout.

Documents centralisés

Accès aux pièces jointes depuis les deux plateformes.

Cohérence

Données toujours à jour et cohérentes entre systèmes.

Données synchronisées

De Dolibarr vers GestMax

Les informations suivantes sont transmises automatiquement :

  • Informations client : Coordonnées complètes du tiers et des contacts
  • Projets/Opportunités : Dossiers commerciaux validés nécessitant un suivi administratif
  • Devis acceptés : Base pour la création du dossier de formation dans GestMax
  • Produits/Services : Catalogue de formations avec durées et tarifs
  • Factures : Éléments de facturation pour rapprochement comptable

De GestMax vers Dolibarr

GestMax renvoie vers Dolibarr :

  • Statuts administratifs : Dossier en cours, validé OPCO, refusé, en attente pièces...
  • Informations financements : Montants pris en charge, reste à charge stagiaire
  • Dates confirmées : Sessions de formation planifiées avec horaires
  • Documents générés : Conventions, attestations, feuilles d'émargement
  • Paiements OPCO : Montants reçus pour rapprochement bancaire (en développement)

Utilisation au quotidien

Pour les commerciaux

Votre workflow dans Dolibarr :

  1. Créez votre opportunité commerciale dans Dolibarr comme d'habitude
  2. Générez et envoyez le devis au prospect
  3. Une fois le devis accepté, marquez-le comme "Accepté" dans Dolibarr
  4. Synchronisation automatique : Le dossier apparaît dans GestMax pour traitement administratif
  5. Suivez l'avancement dans Dolibarr : le statut GestMax s'affiche automatiquement

Visibilité du statut

Dans la fiche projet Dolibarr, un badge indique le statut administratif GestMax : "En attente pièces", "Validé OPCO", "Dossier complet", etc. Vous savez toujours où en est le dossier administratif.

Pour les équipes administratives

Votre workflow dans GestMax :

  1. Récupérez les nouveaux dossiers synchronisés depuis Dolibarr
  2. Complétez les informations administratives (OPCO, financement, dates...)
  3. Générez les documents réglementaires (conventions, attestations)
  4. Mettez à jour le statut : les changements remontent dans Dolibarr automatiquement
  5. Suivez les paiements OPCO et validez la facturation

Bonnes pratiques

  • Saisie complète dans Dolibarr : Plus vos fiches sont complètes, moins il y aura de ressaisie dans GestMax
  • Marquez rapidement les devis acceptés : Cela déclenche la synchronisation vers GestMax
  • Consultez le statut GestMax : Vérifiez régulièrement l'avancement administratif dans Dolibarr
  • Ne dupliquez pas manuellement : Laissez l'intégration créer les dossiers automatiquement
  • Signalez les incohérences : Si vous constatez des données différentes entre les deux systèmes, créez un ticket
  • Respectez les workflows : Ne modifiez pas manuellement les statuts synchronisés
  • Formation continue : Participez aux sessions de formation sur les nouveautés de l'intégration
  • Remontez les besoins : Vos suggestions peuvent influencer les prochains développements

Astuce

Utilisez les champs personnalisés de Dolibarr pour capturer les informations spécifiques à GestMax dès la phase commerciale (type de financement souhaité, OPCO pressenti...). Ces données seront transmises lors de la synchronisation.
