L'Agenda Dolibarr

L'Agenda est le module de gestion du temps dans Dolibarr. Il centralise tous vos événements professionnels : rendez-vous clients, réunions internes, tâches à effectuer, et actions commerciales. L'agenda vous permet de rester organisé et de ne jamais manquer une échéance importante.

Agenda centralisé

L'agenda Dolibarr peut être synchronisé avec votre calendrier Outlook, Google Calendar ou Apple Calendar pour une vue unifiée de tous vos engagements professionnels.

Types d'événements

Rendez-vous

Rencontres avec clients, prospects ou partenaires.

Réunions

Réunions d'équipe, points projets, comités.

Tâches

Actions à effectuer avec date d'échéance.

Appels

Appels téléphoniques planifiés ou à rappeler.

Créer un événement

Étapes de création

1

Accéder à l'agenda

Cliquez sur Agenda dans le menu principal ou utilisez le raccourci dans le tableau de bord.

2

Nouveau événement

Cliquez sur le bouton "+ Nouvel événement" ou directement sur une plage horaire dans le calendrier.

3

Renseigner les informations

Titre, type d'événement, date et heure de début/fin, description, lieu.

4

Lier à un tiers/projet

Associez l'événement à un client, prospect ou projet pour faciliter le suivi.

5

Inviter des participants

Ajoutez les collègues ou contacts externes qui doivent participer.

6

Configurer les rappels

Définissez des notifications email ou popup avant l'événement (15 min, 1h, 1 jour...).

Options avancées

  • Événement récurrent : Créez des événements qui se répètent (quotidien, hebdomadaire, mensuel)
  • Disponibilité : Marquez comme occupé, disponible, absent ou tentative
  • Visibilité : Public (visible par tous) ou privé (visible uniquement par vous)
  • Catégorie : Classez par type (commercial, administratif, formation...)
  • Priorité : Définissez l'importance (haute, normale, basse)
  • Documents joints : Attachez des fichiers pertinents (ordre du jour, présentation...)

Vues de l'agenda

L'agenda Dolibarr propose plusieurs modes d'affichage pour s'adapter à vos besoins :

Vue Jour

Planning détaillé heure par heure de la journée en cours.

Vue Semaine

Aperçu de toute la semaine avec tous vos événements.

Vue Mois

Vue mensuelle synthétique pour planifier sur le long terme.

Vue Liste

Liste chronologique de tous vos événements avec filtres.

Astuce

Utilisez la vue Semaine pour avoir une vision claire de votre charge de travail et éviter les surcharges. La vue Mois est idéale pour anticiper les périodes creuses ou intenses.

Synchronisation et partage

Synchronisation avec Outlook/Google

Synchronisez votre agenda Dolibarr avec votre calendrier habituel :

  1. Accédez aux Paramètres → Mon compte
  2. Dans l'onglet "Agenda", activez la synchronisation CalDAV/iCal
  3. Copiez l'URL de synchronisation fournie
  4. Dans Outlook/Google Calendar, ajoutez un calendrier par URL
  5. Collez l'URL et validez

Synchronisation bidirectionnelle

Les événements créés dans Dolibarr apparaîtront dans Outlook et vice-versa. La synchronisation se fait automatiquement toutes les 15 minutes.

Partage d'agenda

Partagez votre agenda avec vos collègues pour faciliter la coordination :

  • Lecture seule : Vos collègues voient vos disponibilités sans pouvoir modifier
  • Modification : Permet à d'autres utilisateurs d'ajouter des événements dans votre agenda
  • Agendas d'équipe : Créez un agenda commun pour un projet ou un service

Gestion des événements

Modifier un événement

Pour modifier un événement existant :

  1. Cliquez sur l'événement dans l'agenda
  2. Cliquez sur le bouton "Modifier"
  3. Apportez vos modifications
  4. Enregistrez les changements

Événements récurrents

Lors de la modification d'un événement récurrent, vous pouvez choisir de modifier uniquement cette occurrence ou toutes les occurrences futures.

Marquer comme effectué

Une fois un événement terminé, marquez-le comme effectué pour garder un historique propre :

  • Ouvrez la fiche de l'événement
  • Changez le statut de "Planifié" à "Effectué"
  • Ajoutez des notes sur le déroulement si nécessaire

Supprimer un événement

Pour supprimer un événement :

  1. Ouvrez l'événement
  2. Cliquez sur "Supprimer"
  3. Confirmez la suppression

Bonnes pratiques

  • Planifiez vos semaines : Chaque lundi, passez 15 minutes à organiser votre semaine dans l'agenda
  • Liez aux tiers : Associez toujours vos RDV clients à leur fiche tiers pour un historique complet
  • Utilisez les rappels : Configurez systématiquement des rappels pour ne rien oublier
  • Bloquez du temps : Réservez des créneaux pour le travail de fond et les tâches administratives
  • Colorez vos événements : Utilisez des catégories avec codes couleur pour identifier rapidement le type d'événement
  • Préparez vos RDV : Ajoutez dans la description les points à aborder et documents nécessaires
  • Compte-rendu : Après chaque RDV important, ajoutez une note résumant les décisions et actions
  • Synchronisez : Gardez votre agenda Dolibarr synchronisé avec Outlook pour avoir une vue complète
  • Respectez votre temps : Marquez des plages "Ne pas déranger" pour vous concentrer
  • Vérifiez régulièrement : Consultez votre agenda plusieurs fois par jour

Astuce Pro

Créez des événements récurrents pour vos tâches régulières (reporting hebdomadaire, réunion d'équipe, suivi mensuel...). Cela structure votre semaine et vous assure de ne pas oublier ces rendez-vous importants.
